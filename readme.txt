This python script is designed to take the names by state data and derive the following information for the given range of the data:
  - The registered population of the United States per year
  - The number of unique names of the registered population of the United States per year
  - The percentage occupation of the unique names per year
  - The percentage of Male vs Female per year

The program expects the following column of the data in csv format:
State, Sex, Data_Year, Name, Name_Count

The original data set is reteived from the following website:
https://www.ssa.gov/OACT/babynames/limits.html
